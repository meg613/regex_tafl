To design a web application that helps users understand the behavior of regular expressions by:
- Generating strings
- Testing input strings
- Checking equivalence of two regular expressions

**Features**
- Regex Input Palette (supports *, |, ())
- String Generator (displays matching strings)
- String Tester (Accepted / Rejected output)
- Equivalence Checker (compare two regex)
- Modern UI with animations

**Technologies uesd:**
- HTML
- CSS
- JavaScript
